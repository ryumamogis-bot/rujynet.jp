RUJYネットについて

English

RUJYNET is a very simple search engine. It integrates with Google apps and is the most powerful software created by a Japanese child. Let me explain the features of version 1. First, there's the ChatGPT search function. This simply accesses the ChatGPT site when you press a button. Next is the Google search function. You can perform Google searches. The Google search functionality is implemented using JavaScript. Next is Google Maps. The Google Maps function simply loads the Google Maps website. Finally, there's a function to search for today's date. Using Wikipedia, the function to search for today's date is implemented using Javascript. Next is custom search. Custom search can be used by simply replacing the URL location (e.g., https://www.google.com/search?q=here!!) with %s. Please check the homepage for more details.
